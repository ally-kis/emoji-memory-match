# Product Requirements Document (PRD) — PRO MEMORY 50: Cyber-Cat Edition

## 1. Загальний опис проєкту
**PRO MEMORY 50: Cyber-Cat Edition** — це інноваційний гейміфікований веб-додаток для тренування короткострокової пам'яті, концентрації та стресостійкості. На відміну від класичних статичних головоломок, цей продукт пропонує динамічний змагальний ігровий процес у реальному часі проти автономного алгоритмічного супротивника — **Кібера-Кота (Cyber-Cat AI)**, який супроводжується елементами психологічного інтерактиву (тролінгу та мотивації) через інтерфейс.

## 2. Скоуп MVP (Minimal Viable Product)
Для версії MVP було визначено та успішно реалізовано наступний розширений набір функцій:

* **Розширене ігрове поле:** Масив із 50 карток (25 унікальних пар тематичних емодзі), організованих у щільну адаптивну сітку для підвищення рівня складності.
* **Механіка змагання в реальному часі (Real-time AI Match):**
    * Паралельна гра на одному полі: Користувач та Кіт-бот шукають пари одночасно.
    * Візуальний розподіл прогресу: ігрова зона чітко розмежовує успіхи гравця ("Ти") та супротивника ("Кіт").
* **Динамічна система балансу (Фора):** На початку раунду гравець отримує фору в 10 секунд чистого часу, під час яких бот перебуває в режимі очікування (`idle`). Після завершення фори алгоритм бота активується і починає з високою швидкістю закривати картки.
* **Адаптивний ШІ-тролінг (Cyber-Bullying & Praise Mode):** Інтегрована динамічна хмаринка думок Кота-бота. Залежно від дій користувача (серія помилок, швидке знаходження пари або бездіяльність) бот у реальному часі генерує текстові коментарі, що створюють додатковий психологічний тиск та азарт.
* **Ігрові метрики:** Фіксація кількості ходів (Moves), помилок (Mistakes) гравця, загального таймера та інтерактивного прогрес-бару успішності збирання поля у відсотках.
* **Стан перемоги та Екран тріумфу:** Алгоритм перевірки завершення гри, який аналізує, хто зібрав більше пар, та виводить фінальне вітальне вікно з підсумковою статистикою.
* **Кнопка швидкого перезапуску:** Миттєве скидання станів, оновлення таймерів та повторна рандомізація (перемішування) масиву карт.

## 3. Технологічний стек
* **Framework:** Next.js (App Router, React 19) — для оптимізації рендерингу станів карт у реальному часі.
* **Language:** TypeScript — для забезпечення суворої типізації ігрових сутностей та алгоритму бота.
* **Styling:** Tailwind CSS — для розробки сучасного кіберспортивного інтерфейсу із застосуванням Backdrop Blur (ефекту матового скла), кастомних лінійних градієнтів та адаптивності під будь-яку роздільну здатність екрана.
* **State Management:** React Hooks (`useState`, `useEffect`, `useRef`) — для керування асинхронними таймерами фори та кроків штучного супротивника.
