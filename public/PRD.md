# Product Requirements Document (PRD) — Emoji Memory Match

## 1. Загальний опис проєкту
**Emoji Memory Match** — це інтерактивна веб-гра для тренування пам'яті. Мета гри — знайти всі пари однакових емодзі на ігровому полі за мінімальний час та кількість ходів.

## 2. Скоуп MVP (Minimal Viable Product)
Для версії MVP ми визначили наступний набір функцій:
- **Ігрове поле:** 50 карток (25 унікальних пар емодзі), сітка 5x10.
- **Механіка гри:** Переворот карток при кліку. Одночасно можуть бути відкриті лише дві картки. Якщо вони збігаються — залишаються відкритими, якщо ні — закриваються через 0.8 сек.
- **Таймер:** Відлік часу з моменту натискання кнопки "Start Challenge".
- **Лічильник ходів:** Фіксація кожної спроби відкрити пару.
- **Стан перемоги:** Поява вікна з результатом (час та кількість ходів) після відкриття всіх пар.
- **Кнопка Restart:** Можливість скинути прогрес і перемішати картки заново.

## 3. Технологічний стек
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (для адаптивної верстки та анімацій)
- **State Management:** React Hooks (useState, useEffect)
